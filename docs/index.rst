Welcome to grismagic's documentation!
======================================

.. toctree::
   :maxdepth: 2
   :caption: Contents:


``grismagic`` provides grism configuration readers and trace/dispersion
utilities for JWST NIRISS, NIRCam, and Roman WFI grism data.


Usage
=====

.. _installation:

Installation
------------

.. code-block:: bash

    pip install grismagic
